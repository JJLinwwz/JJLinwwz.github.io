婉宁老师教学小助手 — 发给对方 / 上线两种方式
============================================

方式一：只发一个文件（最简单，推荐）
------------------------------------
1. 双击运行「一键打包单文件.bat」
   （或命令行：node build-single.mjs）
2. 会生成「婉宁老师教学小助手.html」（约 0.7MB）
3. 把这个 html 文件通过微信发给对方
4. 对方用手机浏览器打开即可（iPhone 用 Safari，安卓用 Chrome）
5. 建议对方：浏览器菜单 →「添加到主屏幕」，像 App 一样用

注意：单文件版数据仍保存在对方手机浏览器里，换手机前记得点「备份」。


方式二：免费在线链接（你上传一次，她点开就用）
----------------------------------------------
任选下面一种（都免费）：

【A】Netlify Drop（最快，不用写代码）
  1. 先运行「一键打包单文件.bat」生成 html
  2. 打开 https://app.netlify.com/drop
  3. 把「婉宁老师教学小助手.html」拖进网页
  4. 几秒后会得到一个 https://xxx.netlify.app 链接
  5. 把链接发给对方，手机直接打开

【B】Cloudflare Pages
  1. 注册 https://dash.cloudflare.com
  2. Workers 和 Pages → 创建 → Pages → 直接上传
  3. 上传整个 shuxue 文件夹（或只上传打包好的 html）
  4. 部署后得到 xxx.pages.dev 链接

【C】GitHub Pages（链接稳定，适合长期用）
  1. GitHub 新建仓库，上传「婉宁老师教学小助手.html」
  2. 仓库 Settings → Pages → Source 选 main 分支
  3. 访问 https://你的用户名.github.io/仓库名/婉宁老师教学小助手.html

在线版优势：不用发文件，链接收藏即可；语音、麦克风在 https 下更稳定。


开发文件夹说明（不用全发给对方）
--------------------------------
对方只需要：婉宁老师教学小助手.html（单文件）
你自己改功能时用整个文件夹；改完再打包一次即可。
