  双人小屋 · 跨手机实时同步
  画板 + 暖光相簿 · 永久保存 · 不需要 Google 账号
═══════════════════════════════════════

【最快方式：App 内配置】
1. 打开「双人画板」或「暖光相簿」
2. 点「☁️ 连接云端（无需 Google）」
3. 粘贴 Project URL 和 anon key → 保存并连接
4. 两台手机各选「我是婉宁 / 我是 TA」

═══════════════════════════════════════
  第一步：注册 Supabase（用邮箱，不要 Google）
═══════════════════════════════════════

1. 打开 https://supabase.com
2. 点 Sign Up → 用 QQ 邮箱 / 163 邮箱注册（选 Email，不用 Google）
3. 登录后点 New Project → 起名 wanning-xiaowu → 设数据库密码 → 创建
4. 等 1～2 分钟项目建好

═══════════════════════════════════════
  第二步：建数据库表（只需一次）
═══════════════════════════════════════

1. 左侧 SQL Editor → New query
2. 打开项目里的 supabase-setup.sql，全部复制粘贴
3. 点 Run 运行

═══════════════════════════════════════
  第三步：复制配置到 App
═══════════════════════════════════════

1. 左侧 Settings（齿轮）→ API
2. 复制这两项：
   · Project URL（如 https://xxx.supabase.co）
   · anon public key（一长串 eyJ 开头）
3. 在 App 里点「连接云端」粘贴 → 保存并连接
4. 另一台手机同样操作（粘贴相同两项）

═══════════════════════════════════════
  功能说明
═══════════════════════════════════════

· 画板：你画一笔，对方手机几秒内看见（实时同步）
· 画布自动云端存档，下次打开还在
· 「存入回忆」：永久保存到回忆相册，双方可见
· 暖光相簿：传图 + 留言，对方实时看见，永久保存
· 绿点 = 已连接；画板顶部 📡 可打开同步面板

【手机画板】
· 底部固定工具栏：画笔 / 橡皮 / 撤销 / 颜色 / 清空
· 点「工具」展开更多（形状、填色等）
· 点 📡 打开同步、回忆相册、连接云端

【数据持久化】
· 云端：存在 Supabase 数据库，永久保留
· 本机：同时缓存到手机，没网也能看历史

═══════════════════════════════════════
  部署
═══════════════════════════════════════

1. 运行「一键打包单文件.bat」
2. 上传 婉宁老师教学小助手.html 到 Netlify
3. 你和 TA 用同一链接打开

【隐私】roomId 默认 wanning-xiaowu，可改 sync-config.js
有问题把配置（key 打码）发我排查。
