《聊表心意》背景音乐使用说明
================================

代码已支持登录页播放，但 mp3 文件需你自己准备（版权原因无法内置）。

步骤：
1. 用剪映/QQ音乐/网易云等导出《聊表心意》mp3
2. 重命名为：xinyi.mp3
3. 放到本文件夹根目录（与 index.html 同级）
4. 建议大小 3～8MB，过大 GitHub 上传会慢

然后执行：
  git add xinyi.mp3
  git commit -m "add: 登录背景音乐"
  git push origin main

没有 mp3 时：页面照常使用，只是没有背景音乐，不影响登录和其他功能。
